# ANVESHAN-AI-VERSION-2.0
AI powered document authenticity verification system using CNN, OCR, Metadata analysis and ELA forensic layers.
Anveshan AI is an AI-powered document authenticity verification system designed to detect forged, manipulated, and suspicious documents using a multi-layer forensic analysis approach.

With the increasing rise of digital document fraud in banking, government, and business sectors, verifying the originality of documents has become a major challenge. Anveshan AI addresses this problem by combining multiple Artificial Intelligence and forensic techniques to analyze documents from different perspectives.

The system consists of four major intelligence layers:

1. CNN Image Forensics Layer:
A deep learning-based MobileNetV2 model analyzes document images and identifies visual manipulation patterns, inconsistencies, and forgery indicators. The model is trained to differentiate between genuine and forged document samples.

2. OCR Intelligence Layer:
The OCR layer extracts textual information from uploaded documents and helps verify the consistency of document content. It enables future comparison between extracted text and expected document information.

3. Metadata Analysis Layer:
This layer examines hidden document information such as creation details, modification traces, software information, and editing history to identify suspicious activities.

4. Error Level Analysis (ELA) Layer:
ELA detects abnormal compression patterns and highlights regions where image modifications may have occurred. It provides additional forensic evidence for suspicious document analysis.

The outputs from these layers are combined to generate an evidence-based authenticity report, helping users understand whether a document appears genuine or requires further verification.

Key Features:
• AI-based forgery detection
• Multi-layer forensic analysis
• Image manipulation detection
• OCR-based document understanding
• Metadata inspection
• Evidence-based suspicious pattern reporting

Anveshan AI aims to support banks, organizations, and verification systems by reducing manual document checking effort and improving fraud detection capabilities through Artificial Intelligence.
